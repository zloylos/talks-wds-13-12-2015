Модульная система

  • Используется ymaps/modules
  • Позволяет использовать асинхронный require / provide
  • Написал @dfilatov, автор vow

Модульная система: что есть

  • modules.require
  • modules.define
  • modules.isDefined
  • shower.modules = modules.create()

Модульная система: пример

			modules.define('loader', provide => {
				$.script(
					'http://my-plugin-url',
					() => provide(PluginGlobalVar)
				);
			});

Что такое Ymodules: bit.ly/ymodules

Плагины

Плагины: модуль

			shower.modules.define('shower-plugin', [
				'depend'
			], (provide, depend) => {
				…
				provide(plugin);
			});

Плагины: класс

			class MyPlugin {
				constructor(shower, options) {
					…
					shower.container.enterFullscreen();
				}
			}

Плагины: события

			setupListeners() {
				this._shower.player.events
					.on(['next', 'prev'], this._onSlideChange, this)
			}
		
			on(events, callback[, context])
			off(events, callback[, context])
			offAll()
